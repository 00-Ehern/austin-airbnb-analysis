# Austin Airbnb Pricing Analysis
*By Ethan Hernandez | Nov 2025*

## Goal
Clean real Airbnb data and find pricing patterns by room type.

## Key Findings
- Average price: **$168/night**
- Entire homes: **$178** (+130% vs private rooms)
- Top neighborhood: **Downtown**

## Tools
- Python (pandas, matplotlib)
- Dataset: [Kaggle Airbnb Austin](http://data.insideairbnb.com/united-states/tx/austin/2025-10-05/visualisations/listings.csv)
